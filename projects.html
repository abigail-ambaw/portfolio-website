AWS Cloud Monitoring

Implemented a cloud monitoring and alerting system using AWS CloudWatch and SNS to ensure high availability and proactive incident response. The system tracks performance indicators and sends notifications on critical resource usage.

Key Achievements

  • Set up AWS EC2 instances (Ubuntu and Windows) and configured CloudWatch for resource monitoring.
  • Created CloudWatch alarms and integrated SNS for email notifications on performance thresholds.
  • Tested the system for high resource utilization and validated alarm triggers.

Outcome

  • Enabled real-time monitoring and proactive issue resolution through SNS notifications.
  • Improved cloud reliability by monitoring key performance metrics.
